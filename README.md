## Installation

Install my-project with npm

```bash
  cd samcomtechnobrains
```

Composer Install 
```bash
composer install
```

NPM Install 
```bash
npm install & npm run build
```

Setup Env (using .env.example)
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=samcomtechnobrains
DB_USERNAME=root
DB_PASSWORD=
```

Setup Migration 
```bash
php artisan migrate:fresh --seed
```
